# Installation Guide
## Prerequisites
### Hardware Requirements

Before starting, ensure your system has 8GB of RAM. This process is **critical** for *system stability* and `performance`.

> **Note**:
> You must be logged in as a root user to perform these steps.

---

## Configuration Steps
1. Open the terminal.
2. Run the update command.
    * Ensure wifi is connected.
    * Do not close the window.

You can find more details at [SUSE Documentation](https://www.suse.com).
